Through this series of articles we will build an object detector (Viola Jones like) capable of detecting faces.

This tutorial is created to introduce a concept of Viola Jones like object detector, but in step by step manner. The series has the following characteristics:
  1. A moderate knowledge of Python is assumed.
  2. Concepts are introduced gradually and with samples.
  3. A production ready training application is already done for you.
  4. A JavaScript demo is already done so you can try it.
